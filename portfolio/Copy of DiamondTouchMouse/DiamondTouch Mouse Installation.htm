The DiamondTouch Mouse utility is installed by default when the DiamondTouch SDK installer is run. This page describes how to install DTMouse.exe if the DT SDK installer was not run. In any case, DTCalibrate should be run before DTMouse, to calibrate the table to the projected image. If the system Display resolution is changed, then re-run DTCalibrate, and restart DTMouse.

1> Install the DiamondTouch device drivers, as described in the documentation that comes with the DiamondTouch SDK. Adjust the noise levels and the projector image calibration as needed. Touches will not be calibrated to the projected image until DTCalibrate.exe is run. It is recommended to project the computer screen image inside the table (i.e. under-project) enough so that there is a 1/2 inch (1.27 cm) border between the edge of the projected image and the edge of the touch table on all four sides.

2> Register the DiamondTouch ActiveX control. (This is also described in the DiamondTouch SDK documentation.)

cd <DiamondTouch SDK install directory>\bin

regsvr32 .\DiamondTouchEx.ocx

3> Install the "Microsoft .NET Framework version 2.0 " (or later Windows Update. If it has not already been installed, this update can be installed by doing a regular Windows Update and selecting it. Note that this is a non-critical Windows update.

4> Copy the DiamondTouch Mouse directory (DTMouse) to the computer with the DiamondTouch table and execute DTMouse.exe.

To make DTMouse.exe execute every time the computer starts up, put a shortcut to DTMouse.exe in C:\Documents and Settings\All Users\Start Menu\Programs\Startup (or similar).
