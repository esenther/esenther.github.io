Boston Car Data Explorer

    Won Collaborative Data Award in 2014 doing work for the Mass. D.O.T. and MAPC in the "37 Billion Mile Data Challenge". Contributed the Boston Car Data Explorer interactive data visualization tool (1920x1080 version) using D3.js, crossfilter, dc.js. Our submission was the MassVehicleExplorer.

D3.js Elevator Simulator

    Created an interactive visualization of a multi-shaft, double-car elevator scheduling algorithm using first Flash/AS3/Flex then D3.js. Extensive regression testing revealed significant congestion problems with very high traffic loads. Elevator visualization also revealed a "frozen lift" situation that would not have been caught otherwise. See it running here.

Precision Hover

    Invented elegant precision-hover gesture capability (not available in current tablets and touchscreens) that allows touchers to precisely target single pixels on the touch display. Many user experience tasks such as this required many iterations to achieve the desired effect. This work was for the DTMouse mouse emulation utility for the DiamondTouch SDK by Circle Twelve Inc..

DiamondTouch Immersion Distributed Architecture

    Developed WebSockets-based distributed touch capability for video conferencing system to connect remote tables for immersive simultaneous interaction. This was for the DiamondTouch Immersion product.

DiamondTouch Immersion Remote Arms

    Extended multi-user scribbling application to support remote users. Remote users show up as "glowing arms" as they scribble. This was for the DiamondTouch Immersion product.

DTCollaborate for ESRI ArcGIS
